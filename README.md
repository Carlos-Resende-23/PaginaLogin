# 🔐 Página de Login

Uma interface de login desenvolvida com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Campo de e-mail
- Campo de senha
- Botão de login
- Layout centralizado e estilizado
- Estrutura preparada para validações

## 🛠 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 🎯 O que aprendi com este projeto

Neste projeto eu pratiquei:

- Estruturação de formulários com HTML
- Estilização de layouts com CSS
- Organização visual de uma página
- Manipulação básica com JavaScript
- Conceitos de interface de autenticação

## ▶️ Como executar o projeto

Basta abrir o arquivo `index.html` no navegador.
